# Learning Curve -

- Semantic Elements: Utilization of semantic HTML elements to structure different sections of the webpage, providing clarity to the content and improving accessibility.

- Forms: Implementation of HTML forms to gather user input, as seen in the BMI Calculator page, where users can input their weight and height.

- Classes and IDs: Strategic use of classes and IDs to apply specific styles and design to various elements, facilitating a more modular and maintainable code structure.

- Links and Navigation: Incorporation of links for seamless navigation between different sections and pages, creating a cohesive user experience.

- Images and Multimedia: Integration of images and multimedia elements to enhance the visual appeal and engagement of the website.
- Selectors: Effective use of CSS selectors to target and style specific HTML elements.

- Box Model: Application of the box model for precise control over the layout and spacing of elements.

- Transitions: Implementation of transitions to create smooth animations, providing a more polished user interface.

- Layouts (Grid and Flex): Utilization of CSS Grid and Flexbox layouts to achieve responsive and flexible designs.

- Media Queries: Employing media queries to make the webpage responsive, ensuring optimal viewing on various devices and screen sizes.
- Functions: Creation of JavaScript functions, as demonstrated in the script for the BMI Calculator, to perform specific actions or calculations.

- Arrays: Use of arrays for storing and manipulating data efficiently.

- Conditions: Implementation of conditional statements to control the flow of the program, as seen in the JavaScript code for checking if a number is a multiple of 3.

- Loops: Utilization of loops, such as for and while loops, to iterate through data or execute a set of instructions repeatedly.

- DOM Manipulation: Integration of Document Object Model (DOM) manipulation to dynamically update HTML and CSS, creating an interactive and dynamic user interface.


# Challenges -

- Managing multiple HTML files and CSS styles initially led to style conflicts, making it challenging to maintain a cohesive design across different pages.

- Implementing grid-based content presented challenges in maintaining consistency for a seamless user experience.

- Integrating JavaScript functionalities into the project posed challenges, especially when implementing dynamic content, user interactions, and efficient DOM manipulation.
